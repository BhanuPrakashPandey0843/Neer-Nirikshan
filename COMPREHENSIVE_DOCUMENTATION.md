# Neer-Nirikshan Project Documentation

## Project Overview
Neer-Nirikshan is a comprehensive system designed to monitor and analyze water quality parameters. The project aims to provide real-time insights into water conditions, enabling stakeholders to make informed decisions.

## Architecture
The architecture of Neer-Nirikshan consists of multiple layers:
1. **Data Collection Layer**: Utilizes sensors to gather water quality data.
2. **Data Processing Layer**: Processes and filters the collected data for analysis.
3. **Application Layer**: User interface for stakeholders to access data and analysis reports.
4. **Storage Layer**: A robust database for storing historical data and analytics results. 

## Components
- **Sensors**: Various sensors deployed to capture parameters like pH, turbidity, and temperature.
- **Data Processing Unit**: Real-time processing of the incoming data from sensors.
- **Frontend Application**: Web-based interface for visualization and user interaction.
- **Backend Services**: APIs to manage data interactions and business logic.
- **Database**: Stores sensor data, user data, and application logs.

## Work Done
- Established data collection mechanisms with sensors for primary parameters.
- Developed a processing unit to handle real-time data stream.
- Created a web interface for users to visualize data and trends.
- Set up a cloud database for secure data storage.
- Conducted initial testing of components and integration.

## Technical Details
- **Programming Languages**: Python, JavaScript
- **Frameworks**: Flask for backend, React for frontend
- **Database**: MongoDB for unstructured data storage
- **Deployment**: Deployed on AWS for scalability and reliability
